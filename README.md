# HW-12.7-IsKPeriodic
HW-12.7-IsKPeriodic - это домашнее задание 12 модуля курса «Разработчик на C++» от oнлайн-школы SkillFactory.

В HW-12.7-IsKPeriodic реализовано:
- Проверка кратности строки числу K.
- Ввод строки и числа К, с проверкой соответствия условиям задания.
- Реализован простейший интерфейс взаимодействия пользователя с программой.
- Присутствует пользовательское меню.

> Проверка кратности строки числу K проводится в 2 последовательных этапа. 
> В случае получения отрицательного результата, дальнейшие проверки прекращаются.
> 1 этап - проверка теоретически возможной кратности по размеру строки.
> 2 этап - проверка соответствия частей строки подстроке, с шагом, размером с подстроку.
